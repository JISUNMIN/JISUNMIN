안녕하세요  
**4년차 프론트엔드 개발자 지선민입니다.**  
저는 새로운 기술을 빠르게 익히고, 문제를 정확히 진단해 해결하며,  
협업과 재사용성을 고려한 구조적인 개발을 지향하는 개발자입니다.

첫 회사에서 Angular 기반 프로젝트를 경험한 뒤,  
React 환경으로 옮기면서 빠르게 적응하여 여러 Admin·웹 서비스에 적용해왔습니다.  
특히 예약·스케줄과 같이 복잡한 로직의 UI를 구현하며  
트러블슈팅 능력과 상태·데이터 처리 역량을 꾸준히 강화해왔습니다.

또한 풀스택 경험을 바탕으로 백엔드 구조와 API 흐름을 잘 이해하고 있어  
요구사항을 명확히 정리하고 협업하는 데 강점을 가지고 있습니다.  
아울러 공통 컴포넌트를 설계하고 최적화하는 업무를 지속적으로 맡아  
재사용성과 확장성을 고려한 개발 방식으로 팀 생산성 향상에 기여하고 있습니다.

모르는 영역은 스스로 학습하고 정리하는 습관을 가지고 있으며,  
그 과정에서 얻은 인사이트를 Velog에 꾸준히 기록하며 성장해오고 있습니다.
