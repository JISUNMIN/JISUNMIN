안녕하세요.  
**구조 설계와 운영 안정성을 함께 고민하는 4년차 프론트엔드 개발자 지선민입니다.**

Angular 기반 프로젝트로 커리어를 시작해 React 환경으로 전환한 이후,  
여러 Admin 및 웹 서비스를 개발하며 복잡한 비즈니스 로직을 구조적으로 설계해왔습니다.  
특히 예약·스케줄과 같은 상태 의존성이 높은 UI를 구현하며  
데이터 흐름과 상태 관리의 중요성을 체계적으로 다져왔습니다.

Atomic Design 기반 공통 컴포넌트 시스템을 설계하고  
상태 관리 구조를 정비(Redux · Zustand · React Query)하여  
재사용성과 확장성을 고려한 아키텍처를 구축했습니다.  

또한 React Profiler · Lighthouse 기반 성능 개선과  
LogRocket · Highlight.io 도입을 통한 사용자 세션 기반 오류 추적 환경을 구축하며  
운영 환경에서의 문제 대응 속도와 안정성을 함께 개선했습니다.

풀스택 경험을 바탕으로 API 구조와 데이터 흐름을 이해하고 있어  
요구사항을 구조적으로 정리하고 협업 효율을 높이는 데 강점이 있습니다.  
학습한 내용은 Velog에 160개 이상의 기술 글로 기록하며 지속적으로 정리하고 있습니다.
